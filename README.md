# .NET_BlogAPI

A CRUD API for a blog solution that is made with .NET.

Technologies used:
* .NET 6
* Entity Framework Core
* PostgreSQL
* FluentValidation
* Netwonsoft.JSON
* AutoMapper
* Moq (**testing**)
* Bogus (**testing**)
* xUnit (**testing**)

PostgreSQL container is containerized with docker-compose.
